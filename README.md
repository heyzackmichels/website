# zackmichels.com

A website!


### Prerequisites

* [Node.js](https://nodejs.org/en/download/) - open source server environment

## Installing

**1.** Grab directory.

```
git clone https://github.com/heyzackmichels/website.git
```

**2.** At root, install dependencies.

```
npm install
```

## Deployment

**1.** Start app.

```
npm start
```

**2.** Go to [localhost:3000](http://localhost:3000).
