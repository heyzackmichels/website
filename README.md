# zackmichels.com

A website!

## Getting Started

Node and install.

### Prerequisites

* [Node.js](https://nodejs.org/en/download/) - open source server environment

### Installing

Grab directory.

```
git clone https://zackatvt@bitbucket.org/zackatvt/scsszack.git
```

At root, install dependencies.

```
npm install
```

## Deployment

Start app.

```
npm start
```

Go to [localhost:3030](http://localhost:3030).
